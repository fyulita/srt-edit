# srt-edit

A simple tool to adjust .srt subtitles to video.

## Format of a .srt file

```
[dialogue number]
hh:mm:ss,mmm --> hh:mm:ss,mmm
First line of displayed text
n'th line of displayed text

```

### Example 1

```
1
00:02:32,277 --> 00:02:33,903
Rosebud.

```

### Example 2

```
3210
03:46:33,721 --> 03:46:36,765
Frankly, my dear,
I don't give a damn.

```
